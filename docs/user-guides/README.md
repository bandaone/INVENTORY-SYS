## Getting Started

### First-Time Setup

1. **Receive Credentials**: Your Owner will provide your PIN code and role
2. **Download Application**: 
   - Cashiers/Managers: Install POS app on your device
   - Stock Clerks: Access stocktake app via mobile browser
   - Owners: Access dashboard at [dashboard.retailos.com](https://dashboard.retailos.com)
3. **Log In**: Use your 4-digit PIN or email/password
4. **Complete Training**: Follow your role-specific guide

### Security Best Practices

- **Never share your PIN** with anyone
- **Log out** when leaving your workstation
- **Report suspicious activity** to your manager immediately
- **Keep device secure** - lock screen when not in use

## Troubleshooting

### Common Issues

#### "Scanner not working"
1. Check USB/Bluetooth connection
2. Try unplugging and reconnecting
3. Restart POS application
4. Contact IT support if issue persists

#### "Offline mode active"
- Normal during internet outages
- All operations continue as normal
- Data syncs automatically when connection returns
- Check status bar for sync progress

#### "PIN locked out"
- Occurs after 3 incorrect attempts
- Wait 5 minutes before trying again
- Contact manager if you've forgotten your PIN

#### "Garment already sold"
- Garment has been sold previously
- Verify QR code is correct
- Check if label was reused accidentally
- Report to manager for investigation

## Frequently Asked Questions

### General

**Q: What happens if the internet goes down?**  
A: All core operations continue normally using local storage. Data automatically syncs when connectivity returns.

**Q: Can I use my own device?**  
A: POS app works on company-provided Windows/Mac/Android devices. Stocktake app works on any smartphone browser.

**Q: How do I change my PIN?**  
A: Contact your manager or access Settings > Change PIN in the application.

### Cashiers

**Q: What if a customer's mobile money payment fails?**  
A: The POS will display the error. You can retry, switch to cash, or cancel the transaction.

**Q: Can I void a transaction after completing it?**  
A: Only managers can void completed transactions. Call your manager immediately if needed.

**Q: What if the receipt printer runs out of paper?**  
A: Replace the paper roll. The POS will automatically reprint failed receipts when paper is loaded.

### Stock Clerks

**Q: What if I scan an unexpected item?**  
A: It appears in amber category. Complete the stocktake normally - manager will investigate discrepancies.

**Q: Can I pause a stocktake and continue later?**  
A: Yes, your session stays active. Just return to the same URL and continue scanning.

### Managers

**Q: How many labels can I print at once?**  
A: Up to 100 labels per batch. For larger quantities, split into multiple batches.

**Q: What if two locations need the same garment?**  
A: Create a stock transfer request. System tracks all movements with audit trail.

### Owners

**Q: How do I add a new location?**  
A: Dashboard > Settings > Locations > Add New. Requires subscription tier with available capacity.

**Q: Can I export reports?**  
A: Yes, all reports can be exported to PDF or CSV from the Reports section.

**Q: How far back does the audit trail go?**  
A: Complete history since system launch. No data is ever deleted.
